# Browser

## Install Homebrew

https://brew.sh/

```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## Install Yarn

https://yarnpkg.com/lang/en/docs/install/#mac-stable

```
brew install yarn
```

## Install Browser

```
yarn dist
```

Move Browser.app to Application folder
